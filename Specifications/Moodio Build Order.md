# Moodio Clone - Build Order

## Phase 1: Foundation & Data Structure
1. **Data Storage Setup**
   - Local storage or IndexedDB
   - Data models for entries, moods, activities

2. **Basic Mood Entry**
   - 5 default mood levels with colors/icons
   - Create entry with mood + timestamp
   - View today's entry

3. **Activity System**
   - Predefined activity list
   - Select multiple activities per entry
   - Basic activity icons

4. **Entry Management**
   - Add optional text notes
   - Edit existing entries
   - Delete entries
   - List of recent entries

## Phase 2: Navigation & Viewing
5. **Calendar View**
   - Monthly calendar display
   - Show mood on each day
   - Click to view/edit day's entry
   - Navigate between months

6. **Dashboard/Home**
   - Quick entry creation
   - Today's summary
   - Recent entries list
   - Basic navigation menu

## Phase 3: Insights & Analytics
7. **Basic Statistics**
   - Mood distribution chart
   - Mood over time graph
   - Entry count and streaks
   - Activity frequency

8. **Advanced Statistics**
   - Average mood by day of week
   - Mood correlation with activities
   - Date range filtering
   - Monthly summaries

## Phase 4: Customization
9. **User Customization**
    - Customize mood names/colors
    - Add/edit custom activities
    - Theme (light/dark mode)
    - Settings page

## Phase 5: Data Management
10. **Import/Export**
    - Export data (JSON/CSV)
    - Import data
    - Backup functionality

## Phase 6: Polish & Extra Features
11. **Search & Filters**
    - Search past entries
    - Filter by date/mood/activities

12. **Advanced Features** (Optional)
    - Shareable statistics
